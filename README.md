# LazyReader - Image to Audio
This project mainly focused on converting docs/books/newspaper images to audio file for Indian and other regional languages 
## Installation
### requirements
<ul>
<li>python 3</li>
<li>EasyOCR library</li>
</ul>

After creating virtual environment install packages using pip for python
>pip3 install -r requirements.txt
### Run
>python3 manage.py runserver 8000
